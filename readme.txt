Track website visitors and page visits with unique visitor IDs.

== Description ==
Track Website Search helps you monitor visitor activity on your website, including:
* Unique visitor tracking
* Page visit logs
* Admin dashboard reports

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin
3. Visit **Track Website Search** menu
